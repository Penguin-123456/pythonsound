# **使い方**
1.   初期設定の場所を▶を押すか`Ctrl + Enter`を押してください・
2.   durationのところは秒数です。好きな秒数を選んでください。
3.   下のスライダー(hz)を動かしてください。
4.   次に`Ctrl + Enter`を押すか、▶を押してください。
5.   自動で再生されますが、再生できない場合は▶を押してください。
